## Customizing Arcolinux (ArchLinux variant)

### Super key to open whisker menu (xfce)
  * https://github.com/JixunMoe/xfce-superkey
  ```
  git clone https://github.com/JixunMoe/xfce-superkey.git
  cd xfce-superkey 
  make
  sudo make install
  ```
